'Graphs' used for sorting algorithms

  • few_unique.graphml - a twelve-element list of 0's, 1's, and 2's
  • medium_sorting_test.graphml - a seven-element list for testing
  • random_13.graphml - a thirteen-element list; causes quicksort to descend three levels of recursion
  • reverse.graphml - an eleven-element list in reverse order
  • small_sorting_test.graphml - a four-element list; permutation (4,2,1,3); first pivot pick in quicksort is the smallest element
  • sorted_11.graphml - eleven distinct elements (weights 0-10) in increasing order
  • sorted_20.graphml - a twenty-element list, sorted with a few duplicated
  • staggered.graphml - an eleven-element list: [1,3,2,4,3,5,4,6,5,7,6]
