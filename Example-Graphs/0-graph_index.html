Example Graphs

  • createMesh.py - a python script for creating a mesh in graphml format; run standalone (no need to invoke python) with no arguments to get a usage message
  • eight_node_graph.graphml - as the name suggests; this one is unweighted, suitable as a directed graph for the dfs_d algorithm
  • large_sorting_test.graphml - a "graph" with no edges, to be used for sorting algorithms; the weight of each node is the key to be used for sorting; there are eight elements
  • mst_example.graphml - a weighted graph, useful for demonstrating minimum spanning tree algorithms or shortest paths algorithms
  • sorting_test.graphml - a small, four-element, no-edge graph for demonstrating sorting algorithms
  • weighted_example.graphml - a weighted graph, smaller than mst_example, used for demonstrating how Dijkstra's algorithm behaves differently for undirected versus directed graphs
  • geo_40_67_1.graphml - a geometric graph (nodes are positioned as random points on the plane and connected if within a given radius); an example that confounds force-directed layout
  • r_25_40_1.graphml - a random graph with no positional information; useful for testing force-directed layout
